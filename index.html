태그 옵션:

  • maker – 퍼슈트의 제작자를 나타냅니다. 자작 퍼슈트의 경우 "DIY"를 사용하십시오
  • name – 퍼슈트 캐릭터의 이름
  • owner – 퍼슈트의 오너
  • species – 퍼슈트 캐릭터의 종
  • style – 퍼슈트 캐릭터의 스타일 (리얼, 케모, 세미)

예제: "maker:bluefox species:wolf"
이 태그는 Bluefox에서 제작한 늑대 퍼슈트를 검색합니다.

입력 칸을 비우고 검색하면 모든 퍼슈트를 보여줍니다.

Tag options:

  • maker – Creator of the fursuit. Self-made use "DIY"
  • name – Name of the character
  • owner – Name of the fursuit's owner
  • species – Self-explanatory
  • style – Style of the character (real, kemo, semi)

Example tag: "maker:bluefox species:wolf"
Above tag will search for all the wolves made by Bluefox.

Empty the search box to show all of the fursuits.
